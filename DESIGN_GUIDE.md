# CALLING 设计指南

## 设计理念
**深色简洁风格 + 拨号盘UI** - 一个真正懂你的待办清单

### 设计哲学
- **深色系**：深灰色背景（#0F1419），营造沉静、专注的氛围
- **极简主义**：去除冗余装饰，突出内容本身
- **拨号盘UI**：圆形卡片、旋转交互，呼应"CALLING"的呼唤主题
- **微动画**：流畅的滑动、淡入淡出，提升交互质感

### 色彩系统
| 用途 | 颜色 | 说明 |
| :--- | :--- | :--- |
| 背景 | #0F1419 (深灰黑) | 主背景色，深色模式 |
| 卡片 | #1A1F26 (深灰) | 卡片、容器背景 |
| 文字主 | #E8EAED (浅灰白) | 主文字颜色 |
| 文字次 | #9CA3AF (中灰) | 次要文字、描述 |
| 强调色 | #7C3AED (紫色) | 完成状态、强调 |
| 边框 | #2D3139 (深灰) | 分割线、边框 |
| 成功 | #10B981 (绿色) | 完成、连胜 |
| 警告 | #F59E0B (橙色) | 截止日期接近 |

### 字体系统
- **标题字体**：Poppins Bold (700) - 现代、有力
- **正文字体**：Inter Regular (400/500) - 清晰易读
- **数字字体**：Courier New - 等宽，用于计数

### 间距系统
- 基础单位：4px
- 卡片间距：16px
- 栏目间距：24px
- 内容内边距：16px

### 圆角系统
- 小圆角：4px (按钮、标签)
- 中圆角：8px (卡片)
- 大圆角：12px (大容器)
- 完全圆形：50% (拨号盘圆形)

### 阴影系统
- 轻阴影：0 2px 4px rgba(0,0,0,0.3)
- 中阴影：0 4px 12px rgba(0,0,0,0.4)
- 深阴影：0 8px 24px rgba(0,0,0,0.5)

### 动画规范
- 快速过渡：150ms (hover、focus)
- 标准过渡：300ms (页面切换、展开)
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1) (ease-in-out)

## 组件设计

### 拨号盘圆形卡片
- 宽高：120px × 120px
- 背景：#1A1F26
- 边框：1px #2D3139
- 内容居中
- 悬停时：背景变亮 (#252B33)、阴影增强
- 点击时：缩放 0.95

### 输入框 (无感输入纸片)
- 背景：#1A1F26
- 高度：120px
- 圆角：8px
- 占位符文字：#9CA3AF
- 焦点时：边框 2px #7C3AED

### 栏目导航
- 底部固定导航栏
- 5个图标按钮
- 活跃栏目：#7C3AED
- 非活跃：#9CA3AF
- 支持左右滑动切换

### 日历方格
- 宽高：40px × 40px
- 有待办的日子：边框 #7C3AED
- 完成状态：背景 #10B981
- 当前日期：背景 #2D3139
- 字体：12px Inter

## 响应式设计
- 手机优先 (320px 及以上)
- 平板适配 (768px 及以上)
- 桌面优化 (1024px 及以上)
- 手机端：全屏沉浸式
- 桌面端：侧边栏 + 主内容区域布局
